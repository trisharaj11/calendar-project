# 📅 Premium Wall Calendar v3

A sleek, interactive calendar component built with a focus on **UX, responsiveness, and clean frontend architecture**.

---

## ✨ Features

- 📆 Range selection with hover preview  
- 📝 Notes (monthly + date-based)  
- 💾 Persistent data via localStorage  
- 🌙 Dark/Light mode  
- 🎨 Custom monthly themes  
- 📱 Fully responsive  

---

## ⚙️ Tech

- React (Vite)  
- JavaScript  
- CSS
