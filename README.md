# calendar-project
=======
# 🗓️ Premium Wall Calendar v3

## Run
```bash
npm install && npm run dev
```

## What's inside
- 12 unique month palettes (none are common blues/purples/greens)
- Dark/Light mode toggle with smooth transition
- Wall-hanging aesthetic: nail, string, spiral rings
- Cormorant Garamond + JetBrains Mono + Outfit fonts
- Day range selection with hover preview + ripple click
- Notes: monthly + date-scoped, localStorage persisted
- Today marker, weekend accent, note indicators on cells
- Hero image fade-in per month, accent bar, page-slide animation
